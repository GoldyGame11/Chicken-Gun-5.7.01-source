# 🐔 Chicken Gun 5.7.01 — Source Code

> ⚠️ **NOT FOR PRIVATE SERVERS** — For modding purposes only.

## 📖 What is this?
Exported source code of **Chicken Gun 5.7.01**, designed to help modders bring newer game assets into **OpenCG+** by **Nestor**.

## 🎯 Purpose
This repo exists for one main goal:  
Take the original **OpenCG project** and upgrade it with assets from this newer version of the game.

## 🛠️ How to use

### Step 1 — Get OpenCG
Clone or download the original OpenCG project by Nestor.

### Step 2 — Open both projects in Unity
Open this source and your OpenCG+ project side by side.

### Step 3 — Copy what you need
From this repo you can take:
- 🟦 **Meshes** — updated 3D models
- 🎨 **Textures** — new and updated textures  
- 📦 **Prefabs** — ready-to-use game objects
- 🗂️ **Resources** — sounds, configs, other assets

### Step 4 — Paste into OpenCG+
Drop the copied assets into the matching folders in your OpenCG+ project.

### Step 5 — Enjoy your mod! 🎉
